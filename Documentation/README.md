# Village Pack Pro

**Procedural Village Generator for Unity**

Create complete medieval villages with houses, wells, and decorations.

## Features

- **5 Village Styles** - Medieval, Fantasy, Tropical, Oriental, Viking
- **4 Layout Types** - Circle, Grid, Organic, Linear
- **Complete Villages** - Houses, wells, fences, paths, props
- **Atmospheric Lighting** - Warm village lights
- **Customizable** - Adjust house count and size

## Installation

1. Import this package into your Unity project
2. Go to **Tools > Village Pack Pro**

## Usage

1. Select your village style
2. Choose layout type and house count
3. Enable features you want
4. Click "GENERATE VILLAGE"

## Requirements

- Unity 6 LTS or later
- Universal Render Pipeline (URP)
